SSL

I was long thinking about running this blog on my own server, so I can give you some privacy by offering https. But I think the guys over at GitHub have more experience running a server. On the other hand, SSL would probably give a false sense of security here: The blog is made up of static content. With a little bit of profiling it should be easy for an attacker to see what you are reading even with SSL on.
If you want to read over SSL, i suggest to read via https://wamserma.github.io/blog.wamser.eu/.
